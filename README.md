Java Quiz Application:

1.Overview:

This project is a simple **console-based Quiz Application** developed using Java.
The program asks multiple-choice questions to the user and calculates the final score based on the answers given.

It also allows the user to **restart the quiz** using a loop if they wish to continue playing.


2. Features:

* Multiple-choice quiz questions
* User input using Scanner
* Score calculation
* Shows correct answers when the user selects a wrong option
* Option to continue or exit the quiz

3. Technologies Used:

* Java
* Arrays
* Loops (for loop and while/do-while loop)
* Conditional statements (if-else)
* Scanner class for user input

How the Program Works:

1. The program displays a question with four options (A, B, C, D).
2. The user enters their answer.
3. The program checks whether the answer is correct.
4. The score increases for each correct answer.
5. After all questions are completed, the final score is displayed.
6. The user is asked whether they want to continue the quiz or exit.

(i)How to Run the Program:
Step 1: Compile the program

javac QuizApplication.java

 Step 2: Run the program

java QuizApplication
